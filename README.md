<h2>2. Welche Daten werden verarbeitet?</h2>
<p>Diese App verarbeitet <strong>keine personenbezogenen Daten</strong>, die wir speichern oder an Dritte weitergeben. Im Einzelnen:</p>
<ul>
  <li><strong>Gespeicherte Verse:</strong> Wenn du Bibelverse mit „+ Speichern" markierst, werden diese ausschließlich lokal auf deinem Gerät im internen App-Speicher (localStorage des WebView) gesichert. Diese Daten verlassen dein Gerät zu keinem Zeitpunkt.</li>
  <li><strong>Teilen-Funktion:</strong> Wenn du einen Vers über „Teilen ↑" weitergibst, wird der Text direkt an die von dir gewählte App übergeben (z.B. WhatsApp, E-Mail). Wir haben keinen Zugriff auf diesen Vorgang.</li>
  <li><strong>Lernmodus:</strong> Der Fortschritt beim Auswendiglernen wird nicht gespeichert und nicht übertragen.</li>
</ul>

<h2>3. Keine Internetverbindung erforderlich</h2>
<p>
  Alle Bibelverse und die gesamte App-Logik sind vollständig lokal gespeichert. Die App funktioniert
  <strong>ohne Internetverbindung</strong>. Eine Verbindung wird lediglich für das Laden der
  Schriftarten (Google Fonts) beim allerersten Start benötigt – danach werden diese vom Gerät gecacht.
</p>

<h2>4. Keine Werbung, kein Tracking</h2>
<p>
  Diese App enthält keinerlei Werbung, keine Analyse-Tools (kein Firebase, kein Google Analytics,
  kein Crashlytics), keine Drittanbieter-SDKs und keine Social-Media-Einbindungen.
</p>

<h2>5. App-Berechtigungen</h2>
<ul>
  <li><strong>INTERNET:</strong> Wird für das einmalige Laden der Schriftarten (Google Fonts) benötigt. Keine weiteren Netzwerkzugriffe.</li>
  <li><strong>ACCESS_NETWORK_STATE:</strong> Prüft, ob eine Verbindung besteht, um ggf. Offline-Hinweise anzuzeigen.</li>
</ul>
<p>Weitere Berechtigungen (Kamera, Mikrofon, Standort, Kontakte etc.) werden <strong>nicht</strong> angefragt.</p>

<h2>6. Lokale Datenspeicherung</h2>
<p>
  Gespeicherte Bibelverse werden im DOM-Speicher (localStorage) des integrierten WebView abgelegt.
  Du kannst diese Daten jederzeit löschen:
</p>
<ul>
  <li>Android-Einstellungen → Apps → Göttliche Wahrheit → Speicher → Daten löschen</li>
</ul>
